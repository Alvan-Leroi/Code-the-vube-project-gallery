# Tech Store Product Showcase Gallery

This is a simple product showcase website built using **HTML, CSS, and JavaScript**.  
It displays a modern 3-card product gallery with hover effects, fade-in animations, and a light/dark mode toggle.

## Project Overview

The website is designed to look like a clean technology product store. It includes a navigation bar, hero section, featured product cards, a promotional section, and a footer.

## Features

- 3 product cards
- Light and dark mode toggle
- Smooth fade-in animations
- Hover effects on cards and buttons
- Responsive layout for smaller screens
- Apple-style clean design
- Built with no frameworks

## Technologies Used

- HTML
- CSS
- JavaScript
